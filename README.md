# Getting and Cleaning Data Course Project

## Project Description
This project cleans and prepares the Human Activity Recognition Using Smartphones dataset
to produce a tidy dataset suitable for further analysis.

## Files in this Repository
- Run_analysis.R: R script that performs all data cleaning steps
- tidy_data.txt: Final tidy dataset
- Codebook.md: Description of variables and transformations

## How to Run
1. Download and unzip the UCI HAR Dataset
2. Set the working directory to the folder containing "UCI HAR Dataset"
3. Run the script Run_analysis.R
4. The output will be a file named tidy_data.txt
